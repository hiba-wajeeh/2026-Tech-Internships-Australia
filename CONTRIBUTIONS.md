## How to Contribute

Thanks for contributing to this repository! <3  
This repository tracks internship opportunities in Australia and aims to keep listings accurate, useful, and up to date.

---

## Adding an Internship

Please ensure the internship meets the following criteria:

- **Australia-based**
- Relevant to **students or new grads**
- Not a duplicate of an existing listing

### Required Information
When adding or updating an internship, include:

- **Company**
- **Role**
- **Location**
- **Application link**
- **Deadline** (if known)
- **Application Opening Date**
- **Internship Duration**
- **International students**: whether the role accepts or is suitable for international students  
  - Use one of: **Yes / No / Unclear**
  - If unclear, include a source or note if possible

Please ensure all links work and information is accurate.

---

## Submitting a Pull Request

1. Fork the repository
2. Create a new branch for your changes
3. Make your updates
4. Open a Pull Request with a clear and concise description

All pull requests will be reviewed before merging.  
Maintainers may request changes to ensure consistency and accuracy.

---

## Code of Conduct

Please be respectful and constructive in all interactions.  
This is a community-driven project.
